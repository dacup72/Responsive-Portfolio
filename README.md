# Responsive Portfolio

For development, you will only need HTML5 and CSS installed on your environement.

## Install

    $ git clone https://github.com/ORG/PROJECT.git
    $ cd Responsive-Portfolio/

## Languages & tools

### HTML

- HTML for basic page layout.

### CSS
